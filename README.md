# Portfolio/Resume Website

Welcome to the GitHub repository for my personal portfolio and resume website! This site serves as a comprehensive showcase of my professional journey, the projects I've worked on, and the skills I've developed over the years. I created it to provide potential employers, clients, and other industry professionals with a detailed look at my work.

**Please note that this site is currently under development. Not all sections are complete and some data may not be fully presented yet. Please check back for updates!**

## Features

- **About Me:** A brief introduction about me, my career goals, and my interests.
- **Resume:** A detailed listing of my education, work experience, and skills.
- **Contact Me:** A page with my professional contact information and social media links.
- **Tools:** A list of tools which were used for creating current website.

## Technology Stack

This website was built with:

### Main Tools
- TypeScript: A statically typed superset of JavaScript that adds types.
- React.js: A JavaScript library for building user interfaces.
- Webpack: A static module bundler for modern JavaScript applications.
- react-router-dom: The DOM bindings for React Router.
- styled-components: A library for styling React components with tagged template literals.

### Utility Tools
- i18n: An internationalization-framework which provides a way to translate your web application.
- lodash: A JavaScript utility library delivering consistency, modularity, and performance.
- js-cookie: A simple, lightweight JavaScript API for handling cookies.

### Code Quality Tools
- Eslint: A tool for identifying and reporting on patterns found in ECMAScript/JavaScript code.
- Prettier: An opinionated code formatter.

### Package Manager
- npm: The world's largest software registry. Open-source developers use npm to share software.

## How to Navigate

You can navigate through different sections of the website using the navigation bar at the top. Each page corresponds to a different section (About, Resume, Contact, Tools).

## Local Development

1. Clone the repository: `git clone https://github.com/BuHogeJI2/portfolio.git`
2. Navigate to the project directory: `cd portfolio`
3. Open `index.html` in your browser to view the website.
